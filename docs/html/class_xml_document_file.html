Detailed Description

XmlDocumentFile This class handles both saving and loading of GUI settings to an XML-style file. Each element of the audioChain is passed in, and is parsed in put into XML format. Once all elements have been read-in, the file is saved to disk. For loading, the reverse procedure is used. The GUIComponent is responsible for passing in all of the elements of the audioChain on save, and for handling all of the attributes on loading.
