Detailed Description

PulseOximeter This struct has 4 values associated strictly with the pulse-oximeter continuous auditory display, namely, [freqRad] the frequency (as provided by SpO2) in radians, [phase] the current phase of the output sinusoid, [level] the full-scale intensity of the sinusoid, and [period] the total time between heart beats (as provided by HR). The member variable pulseOx holds this information.
