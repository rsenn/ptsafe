Detailed Description

SineAudioSource The main goals of this class are to 1) produce an appropriate sinusoid that corresponds in frequency to the nominal SpO2 level, and corresponds in amplitude to the user-generated level; 2) broadcasts a pulse that corresponds to the nominal heart-rate (in beats per minute); 3) apply the audio effects passed in from the GUI to the signal in a chained fasion.
